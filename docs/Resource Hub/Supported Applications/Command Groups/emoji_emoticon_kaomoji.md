# Emoji, Emoticon, Kaomoji

# Command


| Command                    | Description |
| -------------------------- | ----------- |
| `emoji {user.emoji}`       | eg `emoji dancing` for `💃`                |
| `emoticon {user.emoticon}` | eg `emoticon surprised` for `:o`           |
| `kaomoji {user.kaomoji}`   | eg `kaomoji flower girl` for `(◕‿◕✿)`    |
